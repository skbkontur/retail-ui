## Использование

Используйте поле с датой, когда нужно ввести дату в формате ДД.ММ.ГГГГ. Отличается от обычного поля ввода наличием маски даты.

Если вам нужно поле ввода даты не только с вводом с клавиатуры, но и с выбором в блоке календаря - используйте - [DatePicker](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_date-components-datepicker--docs).
