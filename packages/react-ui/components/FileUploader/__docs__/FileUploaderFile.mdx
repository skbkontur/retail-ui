## Доступность

Строка файла использует [Tooltip](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-tooltip--docs) и [Hint](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-hint--docs). Удаление — фокусируемая кнопка с активацией по `Enter`.

ℹ️ **Полезно:** [Чек-лист доступности](https://tech.skbkontur.ru/kontur-ui/?path=/docs/accessibility--docs)

## Адаптивность

По умолчанию строка файла не меняет свой вид и поведение на мобильных устройствах. Ширина задаётся контейнером `FileUploader` (`width`, `view`).

ℹ️ **Полезно:** [Адаптивность компонентов](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_information-responsive--docs)
