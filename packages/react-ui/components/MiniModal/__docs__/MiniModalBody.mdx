Необязательный блок: если заголовка и кнопок во футере достаточно, тело можно опустить. Наследуемые пропсы — в [Modal.Body](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-modal-modalbody--docs).
