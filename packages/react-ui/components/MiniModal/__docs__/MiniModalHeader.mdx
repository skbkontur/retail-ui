## Использование

По умолчанию в шапке показывается иконка информации. Чтобы оставить только текст, передайте `icon={null}`. Дополнительные пропсы (например `sticky`) наследуются от Modal — см. документацию [Modal.Header](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-modal-modalheader--docs).

Примеры сценариев — на странице [MiniModal](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-minimodal-minimodal--docs).
