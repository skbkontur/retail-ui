# MiniModal.Indent

```jsx
import { MiniModal } from '@skbkontur/react-ui';

<MiniModal>
  <MiniModal.Footer>
    <Button size="medium" use="accent">
      Основное действие
    </Button>
    <MiniModal.Indent />
    <Button size="medium">Вторичное</Button>
  </MiniModal.Footer>
</MiniModal>;
```

<Description />

<Primary />

Вертикальный отступ между группами элементов в зоне `MiniModal.Footer` (на практике — между группами кнопок). Не является частью [Modal](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-modal-modal--docs) — специфичен для мини-модалки.

## Использование

Вставляйте `MiniModal.Indent` внутри `MiniModal.Footer`, когда нужно визуально разделить группы действий (см. пример «Синглтон» на странице [MiniModal](https://tech.skbkontur.ru/kontur-ui/?path=/docs/react-ui_overlay-minimodal-minimodal--docs)).
