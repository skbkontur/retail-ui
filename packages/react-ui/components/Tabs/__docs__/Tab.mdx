## Доступность

Компонент таба по умолчанию имеет семантику ссылки.

**Рекомендации**

- Используйте `aria-label`, чтобы задать доступное имя таба для пользователей скринридеров.
- Используйте `aria-describedby` для связи таба с элементом, содержащим дополнительное, развёрнутое описание.

## Адаптивность

По умолчанию табы не меняют свой вид и поведение на мобильных устройствах.
